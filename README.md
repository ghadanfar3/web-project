# 📰 Article Publishing Platform

A dynamic article publishing web application built using Laravel, allowing users to create, publish, and interact with content in a secure and scalable environment.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register) using Laravel Breeze
- ✍️ Create, edit, and publish articles
- 💬 Commenting system on articles
- 🌐 Public access for browsing articles (guest users)
- 🛡️ CSRF protection for secure form handling
- 📱 Responsive UI using Bootstrap
- ⚙️ MVC architecture for clean and maintainable code

---

## 🛠️ Tech Stack

- Backend: PHP / Laravel
- Database: MySQL
- Frontend: HTML, CSS, Bootstrap
- Authentication: Laravel Breeze
- Version Control: Git & GitHub

---

## 📂 Project Structure

The project follows Laravel's MVC architecture:

- Models → Handle database logic
- Views → User interface
- Controllers → Application logic
